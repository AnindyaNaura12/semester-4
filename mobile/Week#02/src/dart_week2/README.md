1. File bmi.dart
file ini menjawab pertanyaan nomor 1 untuk membuat sebuah program yang dapat menghitung BMI berdasarkan berat dan tinggi yang diinput.
alur dari file ini, Program akan:
    1. Meminta input berat badan dari pengguna (dalam kg)
